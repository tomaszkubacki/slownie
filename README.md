# slownie
slownie is numbers to words converter in polish. It's a port of this [PHP version][PHP version]

## License
MIT License.

## Usage

A simple usage example:

    import 'package:slownie/slownie.dart';
    
    main() {
     print(slownie('44'));
     //printed text: 'czterdzie≈õci cztery'
    }
